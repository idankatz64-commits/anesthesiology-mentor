

## תוכנית: שיפורי UI למסך SetupView

### שינויים

**1. הגדלת כרטיסי הפילטרים** — הסרת `max-w-5xl` מה-container הראשי, הרחבת הגריד כך שהכרטיסים ימלאו את כל הרוחב הזמין. הגדלת padding פנימי של כל כרטיס פילטר.

**2. הגדלת שורות הבחירה ב-dropdown** — הגדלת `max-h-60` ל-`max-h-80`, הגדלת padding של כל שורת אפשרות מ-`p-3` ל-`p-4`, הגדלת גודל הטקסט מ-`text-sm` ל-`text-base`, והוספת separator בין שורות לבהירות.

**3. הוספת אפשרות "מותאם אישית" לבחירת מספר שאלות** — הוספת כפתור חמישי "מותאם אישית" בשורת ה-Session Intensity. בלחיצה עליו מופיע input מספרי שבו המשתמש מזין כמות שאלות. מתחתיו הערה קטנה: "ללא אלגוריתם חכם — בחירה אקראית" כדי לציין שבמצב זה לא מופעל ה-smart selection. בפונקציית `handleStart`, אם נבחר custom — פשוט `startSession(pool, customCount, mode)` ללא `selectSmartQuestions`.
