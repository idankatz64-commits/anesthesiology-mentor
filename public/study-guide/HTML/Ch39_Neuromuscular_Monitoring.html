39. Neuromuscular Monitoring — סיכום לשלב א׳

עקרונות מפתח

  • Objective monitoring חובה: אי אפשר לשלול Residual Neuromuscular Block (RNMB) קלינית בלבד — בדיקה קלינית (הרמת ראש ל-5 שניות, Grip strength) אינה מספיקה; נדרש מוניטור אובייקטיבי (Acceleromyography / Mechanomyography)
  • הגדרת RNMB: TOF Ratio < 0.9 — זהו סף הבטיחות הקליני המקובל כיום; מתחת לסף זה קיים סיכון לסיבוכים נשימתיים
  • Margin of Safety: ניתן לחסום עד 70–75% מהרצפטורים ועדיין לראות תגובה תקינה במוניטור — המוניטור "עיוור" מתחת ל-70% תפוסה
  • חלון הניטור: המוניטור הסטנדרטי רגיש רק לטווח 70%–95% תפוסת רצפטורים (Receptor Occupancy)
  • Phase I Block (Succinylcholine): אין Fade — TOF Ratio נשאר 1.0 גם בשיתוק חלקי; המדד היחיד הרגיש הוא Single Twitch Height
  • Deep Block: כאשר TOF Count = 0, המדד היחיד להערכת עומק השיתוק הוא Posttetanic Count (PTC)
  • RNMB גורם ל: ירידה ב-Chemoreceptor sensitivity להיפוקסיה, פגיעה בשרירי הלוע (Pharyngeal muscles), חסימת נתיב אוויר עליון, ועלייה בסיכון לאספירציה ו-Pulmonary complications

נוסחאות ונקודות חישוב

ספי TOF Ratio — היררכיית ההתאוששות

TOF Ratio משמעות קלינית
< 0.7 Vital Capacity מופחת; Grip strength ~60% נורמה; חולשה קלינית ברורה
0.7 המטופל מסוגל להרים ראש ל-5 שניות (בדיקה לא רגישה מספיק)
0.8 Vital Capacity ו-Inspiratory Force חוזרים לנורמה; עדיין: Diplopia, חולשת שרירי פנים
0.9 שרירי Pharynx ו-Upper Esophagus חוזרים לתפקוד תקין — סף הבטיחות לאקסטובציה
≥ 0.9 ניתן לשלול RNMB קלינית משמעותית

עומק השיתוק — Depth of Block

רמת שיתוק הגדרה מדד מתאים
Intense Block PTC = 0 PTC בלבד
Deep Block PTC ≥ 1, TOF Count = 0 PTC
Moderate Block TOF Count 1–3 TOF Count
Shallow Block TOF Count = 4 + Fade (Ratio < 0.9) TOF Ratio
Minimal/No Block TOF Ratio ≥ 0.9 TOF Ratio

PTC — Posttetanic Count ופרשנותו

PTC משמעות
0 Intense Block — חזרת TOF רחוקה מאוד
1–2 Deep Block — זמן חזרה ממושך
8–10 חזרת TOF קרובה מאוד (Imminent)

פרוטוקול PTC: גירוי טטני 50 Hz × 5 שניות → הפסקה 3 שניות → גירויים בודדים 1 Hz × 15

חלון תפוסת הרצפטורים

תפוסת רצפטורים תגובת המוניטור
0–70% False Normal — תגובה תקינה (Blind Spot)
70–95% שינוי נראה — זהו חלון הניטור האפקטיבי
> 95% Complete Block — TOF Count = 0

נקודות בעלות תשואה גבוהה

⚠️ מלכודות בבחינה — Counterintuitive Facts

  1. הרמת ראש ל-5 שניות ≠ התאוששות מלאה: בדיקה זו אפשרית כבר ב-TOF Ratio 0.7, אך זרימת האוויר העליונה עדיין בסיכון עד TOF > 0.9

  2. Absence of tactile fade ≠ בטיחות: אי-זיהוי Fade ב-TOF, Tetanus, ו-DBS ידנית (tactile/visual) אינו שולל RNMB משמעותי — נדרש מוניטור כמותי (Quantitative)

  3. Succinylcholine — Phase I Block: TOF Ratio = 1.0 גם בשיתוק! הדרך היחידה לכמת: Single Twitch stimulation. DBS ו-TOF ratio יראו "תקין" למרות שיתוק חלקי

  4. Supramaximal Stimulation: יש להשתמש בעוצמת גירוי 20–25% מעל הסף (Supramaximal) כדי להבטיח גיוס כל סיבי השריר ותוצאה ניתנת לשחזור

  5. DBS עדיף על TOF (גישוש ידני): Double Burst Stimulation מזהה Fade ידנית (tactile) טוב יותר מ-TOF, אך שניהם נחותים לעומת מדידה אובייקטיבית כמותית

  6. Inhalational Agents מגבירים שיתוק: גזים נשאפים עושים Potentiation — מאריכים את משך השיתוק ומעכבים התאוששות; TIVA מפחיתה סיכון ל-RNMB

  7. Pancuronium = סיכון גבוה ל-RNMB: תרופות Long-acting (Pancuronium) מעלות משמעותית את הסיכון לעומת Intermediate-acting (Rocuronium, Vecuronium, Atracurium)

  8. Myasthenia Gravis: רגישות מוגברת ל-NDMR (Non-Depolarizing Muscle Relaxants) — מינונים קטנים מאוד עלולים לגרום לשיתוק ממושך

  9. גיל מבוגר + אי-ספיקת כליות/כבד: Clearance מופחת → סיכון מוגבר ל-RNMB; רק Atracurium/Cisatracurium (Hofmann elimination) אינם תלויים באיברים

  10. PTC לא מחליף TOF: PTC מוגדר רק ל-Deep Block (TOF Count = 0); אין טעם לבצע PTC כאשר קיים TOF Count ≥ 1

שאלות נפוצות בנושא

שאלה 1

מטופל עם TOF Count = 0 בסוף הניתוח. איזה מדד ניטור תשתמש בו להערכת עומק השיתוק?

תשובה: Posttetanic Count (PTC) — זהו המדד היחיד שיכול להעריך עומק שיתוק ולנבא את הזמן עד חזרת TOF כאשר TOF Count = 0. PTC 8–10 מרמז על חזרת TOF קרובה; PTC = 0 (Intense Block) מצביע על שיתוק עמוק מאוד.

שאלה 2

מהו ה-TOF Ratio המינימלי הנדרש לאקסטובציה בטוחה, ומדוע TOF Ratio 0.8 אינו מספיק?

תשובה: TOF Ratio ≥ 0.9. בסף 0.8, Vital Capacity ו-Inspiratory Force חוזרים לנורמה, אך שרירי הלוע (Pharynx) ו-Upper Esophagus עדיין לקויים. מתחת ל-0.9 קיים סיכון לאספירציה, חסימת נתיב אוויר, וירידה ב-Hypoxic ventilatory drive. אקסטובציה בטוחה דורשת TOF Ratio > 0.9 בניטור אובייקטיבי.

שאלה 3

מדוע TOF Ratio אינו מועיל לניטור Phase I Block של Succinylcholine, ומה המדד המתאים?

תשובה: ב-Phase I Block, Succinylcholine גורם לדפולריזציה שווה של כל הרצפטורים — אין Fade. לכן TOF Ratio נשאר ≈ 1.0 גם בשיתוק חלקי (כנ"ל DBS). המדד היחיד שמשתנה ליניארית עם עומק השיתוק הוא Single Twitch Height (גובה הטוויץ' הבודד).

שאלה 4

מה הוא "חלון הניטור" של המוניטור הנוירו-מוסקולרי הסטנדרטי מבחינת תפוסת רצפטורים?

תשובה: 70% עד 95% תפוסת רצפטורים (Receptor Occupancy).
- מתחת ל-70%: המוניטור מראה תגובה תקינה — False Normal (Margin of Safety)
- מעל 95%: Complete Block — TOF Count = 0
- רק בטווח 70–95% ניתן לזהות שינוי בתגובה

שאלה 5

מטופל בן 75 עם אי-ספיקת כליות מקבל Pancuronium לניתוח בטן. מהם שני גורמי הסיכון העיקריים ל-RNMB בחדר ההתאוששות?

תשובה:
1. Pancuronium — תרופה Long-acting המסולקת בעיקר דרך הכליות; אי-ספיקת כליות מעכבת פינוי → הארכת משך השיתוק
2. גיל מבוגר + אי-ספיקת כליות — ירידה ב-Renal Clearance + האטה בסירקולציה → הצטברות התרופה

טיפ לבחינה: במצב זה, Atracurium או Cisatracurium (Hofmann elimination, אינם תלויי כליה/כבד) עדיפים לניהול הרדמה בטוחה יותר.
