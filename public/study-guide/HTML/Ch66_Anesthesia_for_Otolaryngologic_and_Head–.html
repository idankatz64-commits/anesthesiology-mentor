66. Anesthesia for Otolaryngologic and Head–Neck Surgery — סיכום לשלב א׳

עקרונות מפתח

  • Difficult Airway — חולי ENT, ובמיוחד חולי סרטן ראש-צוואר, מהווים אוכלוסיית סיכון גבוה לנתיב אוויר קשה. יש לסקור CT או בדיקה אנדוסקופית טרם ההרדמה.
  • Shared Airway — האתגר המרכזי בניתוחי לרינקס: המנתח והמרדים מתחרים על אותו נתיב אוויר. כל טכניקת הנשמה נבחרת בהתאם למיקום הפתולוגיה.
  • ATI (Awake Tracheal Intubation) עם Flexible Bronchoscope — שיטת הבחירה כאשר אינטובציה לאחר אינדוקציה צפויה להיות קשה/בלתי אפשרית. מתבצעת בעדינות ללא כוח, ללא צורך בחשיפת גלוטיס ישירה.
  • Tracheostomy תחת הרדמה מקומית — כאשר ATI אינה בטוחה/מעשית (פתולוגיה נרחבת מדי). במצב חירום — Cricothyrotomy עדיפה (מהירה יותר).
  • NMBAs ו-Nerve Monitoring — בניתוחי Parotid ובניתוחי אוזן הדורשים ניטור עצב הפנים (CN VII): אסור להשתמש ב-Neuromuscular Blockers לאחר האינטובציה (או להשתמש אך ורק בקצרי-טווח ולאפשר התאוששות מוחלטת).
  • Emergence רכה — ביציאה מהרדמה יש למנוע שיעול ו-Straining כדי להימנע מ-Venous engorgement, דימום חוזר והיפרטנסיה, הקריטיים במיוחד לאחר ניתוחי צוואר.
  • Heliox — תערובת הליום-חמצן; מפחיתה צפיפות (Density) הגז ובכך מורידה את התנגדות הזרימה הטורבולנטית בחסימות דרכי אוויר עליונות (Stridor).

נוסחאות ונקודות חישוב

סיווג שברי Le Fort

סוג שם קו השבר סיכון הרדמתי מרכזי
Le Fort I Horizontal / Floating Palate מתחת לאף — Maxillary alveolus בלבד מינימלי לבסיס גולגולת
Le Fort II Pyramidal / Nasomaxillary כולל רצפת Orbit וגשר האף סיכון בינוני ל-CSF leak
Le Fort III Craniofacial Dysjunction / Floating Face דרך Zygomatic arches, Orbits, Nasofrontal suture — הפרדה מוחלטת מבסיס גולגולת סיכון גבוה: Cribriform plate fracture, CSF leak, TBI — אסור Nasal intubation!

⚠️ כלל אצבע: ככל שמספר Le Fort גבוה יותר → קו השבר גבוה יותר בפנים → קרוב יותר לבסיס הגולגולת → סיכון גדול יותר.

פיזיקה של Heliox

פרמטר רלוונטי לזרימה
זרימה למינרית (דרכי אוויר תקינות) תלויה ב-Viscosity (צמיגות) — הליום אינו עדיף
זרימה טורבולנטית (חסימה/Stridor) תלויה ב-Density (צפיפות) — הליום עדיף מאוד
צפיפות הליום ~0.16 g/L (לעומת חנקן ~1.25 g/L)
תערובת Heliox נפוצה 70% He / 30% O₂ (מינימום O₂ = 21%)

⚠️ מגבלה קריטית: ככל שנדרשת FiO₂ גבוהה יותר → אחוז ההליום קטן → אפקט ה-Heliox נחלש.

אנטומיה של Larynx (צילום צוואר לטרלי — מלמעלה למטה)

Hyoid bone (C3)
      ↓
Epiglottis
      ↓
False Vocal Folds (Vestibular folds) — SUPERIOR
      ↓
True Vocal Folds (Glottis) — INFERIOR
      ↓
Subglottis / Trachea

🔑 False (מדומה) = עליון | True (אמיתי) = תחתון

נקודות בעלות תשואה גבוהה

🔴 Airway Management — נתיב אוויר

  • Intermittent Apnea Technique: מנשמים ב-100% O₂ → מוציאים טובוס → מנתח עובד בשדה נקי → סטורציה יורדת → עוצרים, מאנשמים מחדש. החיסרון: קיטוע (Fragmentation) של העבודה הניתוחית לחלונות זמן קצרים.
  • Microlaryngeal ETT: קוטר קטן (5.0 מ"מ) עם Cuff — מאפשר גישה ניתוחית עם הנשמה רציפה.
  • Laser-safe ETT — חובה בניתוחי לייזר בלרינקס (למנוע שריפת הטובוס).
  • Wire-reinforced (Armored) ETT — מונע דחיסה בניתוחים בהם יש לכופף/לסובב ראש.

🔴 Parotidectomy ו-Facial Nerve (CN VII)

  • CN VII חוצה את בלוטת Parotid ומתפצל ל-5 ענפים (Temporal, Zygomatic, Buccal, Marginal Mandibular, Cervical = "Ten Zebras Bit My Chin").
  • ניטור עצב הפנים דורש: היעדר מוחלט של NMBAs בשלב הדיסקציה.
  • Frey's Syndrome (סיבוך מאוחר): הזעה והסמקה של פני הלחי בזמן אכילה — נגרמת מ-Aberrant reinnervation של סיבי ה-Auriculotemporal nerve הפאראסימפתטיים לבלוטות הזיעה העוריות.

🔴 נקודות "מלכודת" (Traps) לבחינה

מצב הציפייה הנפוצה (שגויה) התשובה הנכונה
Le Fort III + טראומה אינטובציה נזאלית Oral intubation בלבד (סיכון ל-Cribriform plate!)
Stridor חמור תמיד לבצע אינטובציה מיידית Heliox תחילה כ-Bridge; ATI בתנאים מבוקרים
Parotidectomy לתת Succinylcholine לאינטובציה מותר לאינטובציה, אך לאסור NMBAs לאחר מכן עד סיום Nerve monitoring
False vs. True Vocal Folds False = תחתון False = עליון (Superior) בכל הדמיה
Heliox — מנגנון משפר viscosity מפחית density (לא viscosity!)

שאלות נפוצות בנושא

שאלה 1:
בחולה עם Stridor חמור משבר גידול סרטני בלרינקס — מה המנגנון שבו Heliox מפחית את עבודת הנשימה?

תשובה:
בחסימת דרכי אוויר, הזרימה הופכת טורבולנטית. בזרימה טורבולנטית, ההתנגדות תלויה ב-Density (צפיפות) של הגז. הליום בעל צפיפות נמוכה פי ~7 מחנקן, ולכן תערובת Heliox (70/30) מפחיתה משמעותית את ההתנגדות ואת עבודת הנשימה.

שאלה 2:
בניתוח Parotidectomy — מדוע יש להימנע ממשתקי שרירים לאורך הניתוח, ומהי ההשלכה המעשית?

תשובה:
המנתח משתמש ב-Nerve stimulator לזיהוי ושמירת CN VII (Facial nerve). נוכחות NMBAs מבטלת את תגובת השריר לגירוי ומונעת זיהוי העצב. לכן: מאנשמים ב-TIVA או Volatile agent ללא NMBAs לאחר האינטובציה, ומוודאים Train-of-Four = 4/4 לפני שלב הדיסקציה.

שאלה 3:
מהו ההבדל בין Le Fort II ל-Le Fort III מבחינת הסיכון להרדמה?

תשובה:
Le Fort II — שבר פירמידלי, כולל Orbit floor ואף; סיכון בינוני ל-CSF leak.
Le Fort III — Craniofacial Dysjunction; הפרדה מוחלטת של שלד הפנים מבסיס הגולגולת; עוברת דרך Cribriform plate → סיכון גבוה ל-CSF leak, TBI, ומעבר ETT נזאלי לגולגולת. אינטובציה נזאלית אסורה לחלוטין ב-Le Fort III.

שאלה 4:
בצילום צוואר לטרלי, מה נמצא מעל (Superior) — False vocal folds או True vocal folds?

תשובה:
False vocal folds (Vestibular folds) נמצאים מעל (Superior) ל-True vocal folds. הסדר מלמעלה למטה: Epiglottis → False VF → True VF (Glottis) → Subglottis.

שאלה 5:
תאר את טכניקת Intermittent Apnea — מה יתרונה ומה חסרונה העיקרי?

תשובה:
יתרון: מעניקה למנתח שדה ניתוחי נקי לחלוטין ללא טובוס (קריטי ב-Posterior glottis).
חסרון עיקרי: Fragmentation — קיטוע הניתוח לקטעים קצרים (כ-1-2 דקות) עם הפסקות חוזרות להנשמה מחדש, הפוגעות בקצב ורצף העבודה הניתוחית.
ביצוע: Pre-oxygenation ב-100% O₂ + TIVA + NMBAs IV → הוצאת ETT → Apnea → עבודה ניתוחית → Sat↓ → הפסקה → Reintubation/Mask ventilation → חזור.
