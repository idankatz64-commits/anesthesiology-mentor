## הדרישות

---

## הפקודות (לפי הסדר)

```bash
# 1. התקנת Python + הכלי uv (אם אין כבר). אם כבר מותקן — לא יקרה כלום רע.
brew install python@3.12 uv

# 2. התקנת Graphify עצמו (שים לב: graphifyy עם שני y)
uv tool install graphifyy

# 3. חיבור Graphify ל-Claude Code (זה השלב שעובד רק על המק)
graphify claude install
```

לאחר מכן, בתוך Claude Code, בתיקיית הפרויקט, מריצים:

```
/graphify .
```

זה בונה את המפה ומייצר 3 קבצים:
- `graph.html` — מפה אינטראקטיבית שנפתחת בדפדפן
- `GRAPH_REPORT.md` — דוח קריא של מבנה הפרויקט
- `graph.json` — נתוני הגרף (ל-AI)
